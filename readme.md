# FooGallery Playground Demo

This is a demo of the FooGallery plugin running in WordPress Playground.

## Usage
[Launch](https://playground.wordpress.net/?blueprint-url=https://github.com/fooplugins/foogallery-blueprint/raw/refs/heads/main/blueprint.json)

## Useful Tools / Resources
[Pootle Playground](https://pootleplayground.com) - Visual Blueprint Generator & Gallery.
[WordPress Blueprints Gallery](https://github.com/WordPress/blueprints/blob/trunk/GALLERY.md) - Gallery of different blueprints on Github.
[Step Documentation](https://wordpress.github.io/wordpress-playground/blueprints/steps) - Official documentation for blueprint steps.
[Blueprint Builder](https://playground.wordpress.net/builder/builder.html) - Playground Blueprint Builder (best way to test blueprints).
